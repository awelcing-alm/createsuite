Quick Start Guide

  • cs init --name my-project --git
    Initialize a new CreateSuite workspace with git tracking
  • cs agent create alice --capabilities "frontend,testing"
    Create an agent with specific capabilities
  • cs task create --title "Build feature" --priority high
    Create a new task with priority level
  • cs agent assign cs-abc12 alice
    Assign a task to an agent for execution
  • cs convoy create "Feature X" --tasks cs-abc12,cs-def34
    Group related tasks into a convoy
  • cs status
    View workspace status and statistics
